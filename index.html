HTML Cheatsheet
Tag Name Description
<h1-h6> Heading Elements These elements represent six levels of section headings. With h1 being the highest section level and h6 being the lowest
<p> Paragraph Element Paragraphs are usually represented in visual media as blocks of text separated from adjacent blocks by blank lines and/or first-line indentation
<img> Image Embedded Element The img html element embeds an image into the document
